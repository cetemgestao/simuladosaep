🔧 Configuração do Sistema

Para o simulado funcionar corretamente, você precisa configurar o Google Apps Script e publicar como Web App. Siga os passos abaixo:

  1. Abra sua Planilha Google com as abas Questoes e Resultados
  2. Vá em Extensões → Apps Script
  3. Cole o código do arquivo PLANILHA.gs fornecido
  4. Clique em Implantar → Nova implantação → Web App
  5. Configure: Executar como Eu, Acesso Qualquer pessoa
  6. Copie a URL gerada
  7. Cole a URL no campo abaixo e no arquivo index.html
